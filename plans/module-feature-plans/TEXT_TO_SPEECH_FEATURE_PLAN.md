# Text To Speech Feature Plan

## Goal
- improve text-to-speech into a richer audio output tool

## Planned Feature Groups

### Voice Controls
- favorite voices
- voice presets
- speed control
- tone control

### Content Features
- single speaker mode
- multi-speaker script mode
- pronunciation guide support
- reusable script presets

### Output Features
- MP3 export
- WAV export
- audio history

## Phases

### Phase 1
- favorite voices
- speed control
- better downloads

### Phase 2
- script presets
- audio history

### Phase 3
- pronunciation controls
- advanced multi-speaker studio mode
