# Live Conversation Feature Plan

## Goal
- make real-time voice conversation stable, clear, and practical

## Planned Feature Groups

### Connection Reliability
- reconnect handling
- connection status indicator
- microphone permission guidance
- graceful error recovery

### Conversation Experience
- language selection
- voice selection
- speaking indicator
- mute and pause controls

### Output Features
- transcript save
- transcript export
- session summary
- action item extraction

## Phases

### Phase 1
- status indicators
- reconnect flow
- transcript save

### Phase 2
- language controls
- transcript export
- session summary

### Phase 3
- meeting mode
- interview mode
- assistant voice presets
