# Workflow Builder Feature Plan

## Goal
- make the workflow tool editable, reusable, and export-friendly

## Planned Feature Groups

### Generation Features
- workflow presets
- category-based prompt templates
- better goal guidance

### Editing Features
- edit generated steps
- reorder steps
- delete or add steps
- save workflow versions

### Output Features
- TXT export
- CSV export
- PDF export
- DOCX export

## Phases

### Phase 1
- edit steps
- reorder steps
- export improvements

### Phase 2
- templates
- saved workflows

### Phase 3
- collaborative checklist mode
- workflow library
