# Low-Latency Chat Feature Plan

## Goal
- provide the fastest and simplest chat experience in the app

## Planned Feature Groups

### Speed-Oriented UX
- streaming response mode
- fast retry
- minimal UI mode
- quick prompt presets

### Lightweight Productivity
- short answer mode
- bullet answer mode
- translation quick mode
- instant rewrite mode

### Session Features
- temporary history
- one-click clear
- keep recent prompts

## Phases

### Phase 1
- streaming responses
- quick presets
- recent prompts

### Phase 2
- response format controls
- lightweight history

### Phase 3
- fast utility modes
- quick reusable preset packs
